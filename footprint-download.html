What Footprint does
Four shields. One extension.
🔍

URL & Site Scanner

Every page you visit is automatically checked against VirusTotal's database and scanned for phishing keywords. Get an instant risk rating before you type a single password.

🖼️

AI-Resistant Image Protection

Add a customizable visible watermark, embed hidden ownership information and apply experimental pixel-level perturbations. Footprint helps discourage unauthorized reuse and provides verifiable evidence of image ownership.

👤

Username Similarity Checker

Compare a trusted official username with a suspicious account. Footprint analyses character changes, number substitutions, separators and look-alike Unicode letters, then provides an explainable impersonation risk score.

🛡️

Cyber Hygiene Dashboard

Review your browsing safety habits using your website scan history. Footprint calculates an estimated cyber-hygiene score, identifies repeated risky domains and provides personalised recommendations.

How it works?
Up and running in 3 steps
1

Install the Chrome extension

One click from the Chrome Web Store. No account needed, no data collected.

2

Browse normally

Footprint runs quietly in the background. The icon turns red if something looks off, green if you're safe.

3

Use the tools when you need them

Click the extension to watermark images, check usernames, see your full scan history or cyber hygiene score on this dashboard.
